
# Telecom Customer Churn Prediction

Predict whether a customer will churn using historical behaviour data.
## Data Source

This project is based on [Telco Customer Churn: IBM Dataset](https://www.kaggle.com/datasets/yeanzc/telco-customer-churn-ibm-dataset).

## Virtual Environment Setup

1. Create a virtual environment.
```bash
  python -m venv venv
```

2. Activate the virtual environment.
- For Windows
```bash
  venv\Scripts\activate
```
- For MacOS
```bash
  source venv/bin/activate
```
3. Install the dependencies.
```bash
  pip install -r requirements.txt
```
